XR-Lang

The love child of a Lisp Machine and a Smalltalk environment, raised in VR

Programming hasn't fundamentally changed since the 1970s. We still write text files and compile them. We still debug by printing statements. We still treat code as something separate from the systems it creates.

XR-Lang changes that. It's a programming language where the code, the environment, and the created world are one living system. Where you can reach out and touch your data structures. Where time itself becomes a debuggable dimension.
